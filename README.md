# GenLayer Validator Research

## Overview
This research explores key metrics for monitoring GenLayer validators.

## Metrics
- Validator uptime
- Latency
- TPS
- Block response time

## Findings
Validator uptime is the most important factor for network reliability.

Latency affects smart contract execution performance.

TPS helps measure overall throughput.

## Goal
Help builders understand network health.
